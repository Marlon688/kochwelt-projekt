
# Kochwelt

Kochwelt is a simple recipe website created as part of a collaborative learning project.

The goal of this project is to practice working in a team while building a small web application using HTML, CSS, and JavaScript.

---

## Project Goal

The website presents different recipes and basic cooking information.

Main features include:

- Homepage
- Recipe overview
- Individual recipe pages
- Responsive layout
- Basic interactive elements with JavaScript

---

## Tech Stack

This project uses basic web technologies:

- HTML
- CSS
- JavaScript

No frameworks or external libraries are used.

## Project Structure

- **html/** - html files
- **css/** – stylesheets

## How to Run the Project

1. Clone the repository
2. Open `index.html` in your browser

No installation or build process is required.
