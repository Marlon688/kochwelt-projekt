
# Kochwelt – Design Guidelines

This document defines basic design rules to keep the website visually consistent across the project.

---

## Spacing

To keep the layout consistent, we use a fixed spacing system.

### Mobile spacing

Small spacing: 8px  
Standard spacing: 16px  
Large spacing: 24px  

### Desktop spacing

Small spacing: 16px  
Standard spacing: 32px  
Large spacing: 64px  

These values should be used for margins, paddings, and gaps whenever possible.

---

## Responsive Breakpoints

Mobile:
0 – 768px

Tablet:
768px – 1024px

Desktop:
1024px+

Layouts should adapt accordingly.

---

## Typography

Main font:
(schreibt ihr rein)

Headings:
H1 – main page titles  
H2 – section titles  
H3 – smaller titles  

Paragraph text should remain readable on all devices.
